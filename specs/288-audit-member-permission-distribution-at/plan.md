# Implementation Plan: Audit Member Permission Distribution at Org Level

**Branch**: `288-audit-member-permission-distribution-at` | **Date**: 2026-04-20 | **Spec**: [spec.md](./spec.md)

## Summary

Surface a role-distribution panel (admin / member / outside collaborator counts + percentages) on the org Governance tab. Flag orgs where the admin ratio exceeds a configurable threshold (provisionally 10%, tracked for calibration in issue #152). Admin count is shared from the existing Stale Admins panel to avoid a duplicate API call. New REST fetchers are added for role=member and outside_collaborators.
